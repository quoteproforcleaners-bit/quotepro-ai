The agent should not just send all data into an LLM blindly.
Use a tool / function-calling architecture.

Implement tools such as:

1. get_customer_summary
2. search_customers
3. search_leads
4. search_quotes
5. get_quote_details
6. get_pipeline_summary
7. get_conversion_metrics
8. get_revenue_metrics
9. get_pricing_profile
10. get_stale_quotes
11. get_reactivation_opportunities
12. get_recent_activity
13. get_service_type_breakdown
14. get_top_customers
15. get_lost_quote_reasons
16. create_follow_up_draft
17. suggest_sales_script
18. search_knowledge_base
19. search_industry_knowledge
20. generate_action_plan

Design the system so the LLM can call the appropriate tool(s), then synthesize an answer.
