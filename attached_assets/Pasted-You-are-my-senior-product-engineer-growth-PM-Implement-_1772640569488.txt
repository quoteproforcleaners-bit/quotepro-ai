3) PRICING AUDIT ALERT (MARGIN/HOURLY PROTECTION)
========================================================
A) Settings
Add “Pricing Guardrails” in Settings:
- Target $/hr (default $60)
- Target margin % (optional)
- Minimum job price (default $179)
- Typical travel buffer (minutes) optional
- Toggle: “Show pricing audit warnings” default ON

B) Audit Logic
On quote results screen, compute:
- estimated hours (if app collects time estimate; if not, infer based on sqft/beds/baths using a simple baseline table and allow user override)
- implied $/hr = quote_total / estimated_hours
If implied $/hr < target:
- show warning banner:
  “⚠️ This quote puts you at ~$42/hr (target $60/hr).”
Provide 1-tap fixes:
- “Raise to Minimum” (sets to min price if below)
- “Add common upsell bundle (+$X)”
- “Adjust time estimate” (opens quick edit)
Also flag if quote_total < minimum job price:
- “⚠️ Below your minimum job price ($179).”
