Build a major new QuotePro AI feature called “Live Job Updates” that allows app users to generate and send a customer-facing update page for any job.

OBJECTIVE
Create a premium customer experience where the cleaning business can send a live service update link to the customer. The customer should be able to view job details, current status, checklist progress, completion photos, and a service timeline in real time as the cleaner or app user updates the job.

CORE PRODUCT REQUIREMENTS

1. JOB DETAILS SCREEN ENHANCEMENT
In the Jobs section, when the app user opens any job, add a new prominent button:
“Send Update Page”

Also add a new section or tabbed interface inside the selected job with these tabs:
- Overview
- Progress
- Checklist
- Photos & Notes

2. CUSTOMER UPDATE PAGE LINK
When the app user taps “Send Update Page”:
- Generate a unique secure public-facing URL for that job
- Example structure: /job-updates/:jobId/:token
- The page must not require login
- Use token-based access so the link is hard to guess and secure
- The link should open a branded, mobile-friendly customer page

3. CUSTOMER-FACING PAGE CONTENT
The customer-facing page should display:
- Company name and logo
- Job date
- Service type
- Arrival window
- Assigned cleaner or team name if available
- Current job status
- Progress percentage
- Checklist sections and completed items
- Uploaded photos
- Service notes marked as customer-visible
- Real-time timeline of updates
- Final “Job Completed” state

Statuses supported:
- Scheduled
- En Route
- Service Started
- In Progress
- Final Touches
- Completed

4. JOB PROGRESS UPDATING
Inside the app, in the Progress tab for each job, create quick action buttons for status updates:
- Mark En Route
- Mark Service Started
- Mark In Progress
- Mark Final Touches
- Mark Completed

When tapped:
- Update job status
- Add timestamped entry to customer-facing timeline
- Recalculate progress percentage
- Sync immediately to the customer-facing page

5. CHECKLIST SYSTEM
Inside the Checklist tab:
- Support room-based or task-based checklist groups
- Each checklist item should have a toggle checkbox
- Show completed vs total count
- When items are checked off, update progress on customer page
- Allow template-based checklist defaults based on job type

Example checklist groups:
- Kitchen
- Bathrooms
- Bedrooms
- Common Areas
- Final Touches

6. PHOTOS AND NOTES
Inside Photos & Notes tab:
- Allow image upload from mobile device
- Store photos per job
- Display uploaded photos on customer-facing page
- Support note creation with a toggle:
  - Internal only
  - Visible to customer

7. SEND UPDATE PAGE WORKFLOW
When “Send Update Page” is tapped, open a modal with options:
- Send by SMS
- Send by Email
- Copy Link

Generate AI-written message suggestions for both SMS and email using business name and customer first name.

Example SMS:
“Hi [First Name], here is your live service update page for today’s cleaning. You can view job progress, checklist updates, and completion photos here: [link]”

Example Email subject:
“Your live cleaning service update page”

Example Email body:
“Hi [First Name], your service update page is ready. You can use the link below to view service details, progress updates, and completion photos in real time: [link]”

8. REAL-TIME SYNC
Ensure that when the cleaner or app user updates the job:
- customer page refreshes automatically or polls every few seconds
- timeline updates are reflected quickly
- status, checklist, and photos stay in sync

9. FINAL COMPLETION AUTOMATION
When a job is marked Completed:
- Add a final timeline entry
- Show a completion state on the customer page
- Prompt app user to send a final completion message by SMS or email
- Include link to the final update page

10. UI / UX STYLE
Design should feel premium, modern, and simple.
Use clean cards, progress bars, checklist toggles, timeline components, and mobile-first layout.
This should feel like a polished customer trust feature, not an internal admin tool.

TECHNICAL REQUIREMENTS
- Create database fields and models needed for:
  - public update token
  - job status history
  - checklist items and groups
  - photo records
  - customer-visible notes
  - message send log
- Preserve existing job data structure
- Secure public links with randomized token access
- Build this so it can scale later into push notifications and automated workflows

DELIVERABLES
- Updated Job Details screen
- Send Update Page modal
- Customer-facing public job update page
- Progress tab with quick actions
- Checklist tab
- Photos & Notes tab
- Timeline component
- SMS/email AI message generator
- Back-end data model updates
- Full mobile responsive UI

IMPORTANT PRODUCT INTENT
This feature is not just proof of completion.
It is a “Live Job Updates” experience that builds trust, reduces customer anxiety, and differentiates QuotePro AI as premium cleaning business software.
