Please update the QuotePro welcome / onboarding screen to eliminate any confusion that the dollar amounts shown are the price of the app.

PROBLEM TO SOLVE
Right now, the Good / Better / Best boxes with prices like $180 / $240 / $320 may be interpreted by new users as QuotePro subscription pricing instead of a sample quote. That confusion could be causing users to hesitate or abandon the flow.

GOAL
Make it instantly obvious that these are EXAMPLE customer quote options generated by the app, not the cost of QuotePro.

WHAT TO CHANGE

1. ADD CLEAR CONTEXT LABEL ABOVE THE PRICE BOXES
Add a small but highly visible label above the Good / Better / Best example pricing cards.

Use one of these options, prioritized in this order:
Preferred:
- Example Quote Your Customer Sees

Other acceptable options:
- Sample Cleaning Quote
- Example Quote Options
- Sample Quote Generated in 30 Seconds

This label should sit directly above the Good / Better / Best cards and visually connect to them.

2. ADD SUPPORTING MICROCOPY UNDER THE LABEL
Add a short explanatory line under the label, such as:
- These are sample customer-facing quote options, not app pricing.
OR
- Example pricing for a cleaning job generated inside QuotePro.
OR
- Your customers see polished Good / Better / Best quote options like this.

Choose the version that feels most premium and natural inside the existing UI.

3. MAKE THE GOOD / BETTER / BEST SECTION FEEL LIKE A DEMO QUOTE
Adjust the styling and/or layout slightly so it feels like a sample output rather than product pricing tiers.

Possible improvements:
- Add a small “Sample” badge above or near the cards
- Add a subtle caption like “3 bed / 2 bath home”
- Add “Most Popular” badge to Better if it improves clarity
- Slightly de-emphasize the dollar amounts relative to the section label so the context is understood first

Do NOT make it cluttered. Keep the same premium, minimal feel.

4. IMPROVE THE HEADLINE / SUBHEAD IF NEEDED
Current message is about closing more cleaning jobs in 30 seconds. Keep that overall direction, but make sure the message helps connect the pricing boxes to quote creation.

Possible improved versions:
Headline:
- Close More Cleaning Jobs in 30 Seconds
Subheadline options:
- Create polished Good / Better / Best quotes your customers can approve fast
- Generate professional quote options in seconds
- Turn job details into customer-ready quotes instantly

Use whichever version best supports conversion and clarity.

5. UPDATE CTA LANGUAGE
Review the main CTA:
Current:
- Create My First Quote

Change to:
Preferred:
- Create Your First Quote

Acceptable alternatives:
- Generate Your First Quote
- Build Your First Quote

Reason: “Your” is clearer and more natural than “My” in this context.

6. KEEP THE SIGN-IN OPTION CLEAR
Do not remove:
- Sign In to Your Account

But make sure the primary CTA remains visually dominant.

7. MAINTAIN EXISTING PREMIUM VISUAL STYLE
Keep:
- dark theme
- blue accent color
- multilingual selector
- clean spacing
- simple onboarding flow

This is a refinement, not a redesign.

8. MOBILE-FIRST
This screen must remain excellent on mobile.
Check:
- text wrapping
- spacing between label, subtext, and quote cards
- button sizing
- visual hierarchy
- no crowding

9. OPTIONAL IMPROVEMENT
If it looks good, add a tiny line below the cards or near the CTA:
- Free to try
- No complicated setup
- Save quotes and send to customers
Only include this if it improves conversion without adding clutter.

10. FINAL STANDARD
The new user should understand within 2 seconds:
- this app helps create quote options for cleaning jobs
- the dollar amounts are sample quote outputs
- the main next step is to create their own quote

Please implement the update directly in the current onboarding/welcome screen using the app’s existing design system and components. Preserve the premium aesthetic while improving clarity and conversion.
