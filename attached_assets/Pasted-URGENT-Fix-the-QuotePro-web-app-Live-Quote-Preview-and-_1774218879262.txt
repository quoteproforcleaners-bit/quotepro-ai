- every 1000 square feet = $85 base price

- every bathroom = $18

Interpretation for MVP/default behavior:

- bedrooms add $15 each
- bathrooms add $18 each

Implement square footage logic clearly and consistently.
Example default behavior:
- 1000 sq ft = $85
- 2000 sq ft = $170
- 2300 sq ft = 2.3 x $85 if using proportional pricing
OR a clearly documented rounding strategy if the app uses rounded increments

Pick one consistent method and document it.
Recommended: use proportional calculation unless the existing product already has an intentional rounding rule.

IMPORTANT:
These values are defaults only.
They must be editable in Pricing Settings by the software user.
