Build these features:

1. Unique Public Quote Request URL Per User
Each logged-in app user should get a unique public URL like:
- /request/{slug}

Examples:
- /request/sparkle-cleaning
- /request/mikes-maids

Requirements:
- add a field on each user/business record for a unique public slug
- generate a slug automatically from business name if one does not exist
- sanitize slug: lowercase, hyphens, no special characters
- if slug is taken, append a short unique suffix
- allow the user to edit their slug from settings
- validate uniqueness before saving
- allow the user to enable or disable their public request page

Suggested database fields:
- public_quote_slug
- public_quote_enabled
- public_quote_button_text (optional)
- public_brand_color (optional, only if easy)
