PART A — GROWTH AUTOMATION FEATURES (BUILD ALL)
========================================================

A1) Review + Referral Automation Engine (HUGE)
Trigger: When a job is marked “Completed” (or manually “Complete Job”)
Flow:
- Step 1 (2 hours after completion OR next morning): “How did we do?” (1–5 rating)
- If rating >= 4: prompt to leave Google review (user configures review link)
- If rating <= 3: route to private feedback message (avoid public bad review)
- Step 2 (48 hours after positive rating): “Know anyone who needs a cleaner?” referral ask
Requirements:
- Template library (friendly, professional, short)
- Per-customer do-not-contact toggle
- Guardrails: max 1 review request per X days per customer (configurable)
- Logging: requested / responded / review_clicked / referral_sent

A2) Auto Rebook System (Retention)
Segments:
- One-time customers: nudge after 21–35 days (configurable)
- Recurring customers who “missed” a cycle: nudge after expected cadence + grace period
Implementation:
- Create “Rebook Tasks” in a queue with one-tap SMS/email compose
- Optional “Book Now link” field (if user has booking link) merge into message
- Outcomes: rebooked / not interested / snoozed

A3) Upsell Trigger Engine (LTV)
Triggers:
- Deep clean due every N months since last deep clean (default 6)
- Add-ons every N visits (e.g., fridge every 3 visits, oven every 6)
- Upgrade frequency suggestion (monthly → biweekly) based on home size or customer notes (rule-based)
Implementation:
- “Upsell Opportunities” screen with suggested upsell + 1-tap outreach
- Messages include merge fields and optional AI rewrite
- Log outcomes: accepted upsell / declined / snoozed

A4) Abandoned Quote Recovery (Close Rate)
Triggers:
- Quote Sent and no follow-up after 24h → create follow-up task
- Quote Sent and stale 3 days → “value reminder” task
- Quote Declined or Expired within last 180 days → “revisit offer” task
Implementation:
- Task queue + notifications + one-tap compose
- “Win Probability” simple scoring to prioritize tasks:
  - Age, quote amount, lead source, last touched, customer responsiveness (manual flag)
- Guardrails: 1 follow-up/day per quote; max 3 total follow-ups (configurable)

A5) Dormant Revenue / Reactivation Campaigns
Identify:
- Customers inactive 60/90/120 days (filter)
- Lost quotes last 180 days
- One-time customers never recurring
Provide:
- “Dormant Revenue” dashboard: counts + estimated recoverable $
- “Campaign Builder (Lite)”
  - select segment + channel + template
  - creates a batch of tasks (assisted automation)
  - optional connected send stub
- Estimation:
  - dormant customer: avgTicket * 0.25
  - expired quote: total * 0.20
  - declined quote: total * 0.10

A6) “Marketing Mode” Toggle (One Switch)
Add to Automations Hub:
- “Turn on Smart Marketing”
When enabled, it turns on:
- Abandoned Quote Recovery tasks
- Weekly Reactivation prompt
- Review request workflow (post-job)
- Referral ask workflow (post-review)
- Rebook nudges
- Upsell triggers
All with safe defaults + quiet hours + max sends/day.

A7) Revenue Forecast + Insights (Differentiator)
Forecast:
- Based on open quotes * close rate (rolling) + accepted jobs (if tracked)
- Display confidence band (simple)
Insights:
- “Revenue at risk today: $X”
- “Oldest quote: Y days”
- “Your follow-up speed vs last week”
- “Top 3 actions to increase revenue this week”
Add: Weekly recap push + screen.

A8) Simple “Marketing Assets” Generator (Optional but valuable)
Inside app:

Keep it simple and polished, not Canva.
